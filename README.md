# flutter, dart, graph-algorithms, dijkstra, bellman-ford, prim, kruskal, shortest-path, minimum-spanning-tree, algorithm-visualization, mobile-app
A Flutter-based mobile app for visualizing classic graph algorithms interactively. 
Users can build custom graphs by adding nodes and edges (directed or undirected), 
then run Dijkstra, Bellman-Ford, Prim, or Kruskal with step-by-step animation and 
detailed result tables. Supports adjustable animation speed, source node selection, 
and displays shortest paths and MST totals in real time.
